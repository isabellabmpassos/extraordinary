Extraordinary Trip é um site de pacotes de viagem, criado com o objetivo de oferecer experiências únicas de viagem para diferentes destinos.

-Período de Desenvolvimento
Este projeto foi desenvolvido entre os dias 20 de agosto e 4 de setembro de 2024. Ainda está em construção.

-Tecnologias Utilizadas
HTML: Estrutura do site
CSS: Estilização e layout

-Funcionalidades
Exibição de pacotes de viagem para diversos destinos.
Design responsivo, proporcionando uma boa experiência em diferentes dispositivos.
